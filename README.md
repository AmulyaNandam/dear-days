
# 🌊 DearDays  
### A Minimalist Cinematic Productivity & Memory Hub  
DearDays is a thoughtfully designed productivity and journaling platform built using the MERN stack. It combines elegant UI design with smooth interactions to help users manage tasks, capture thoughts, and reflect on daily experiences — all in one calm, distraction-free space.
---
## ✨ Key Features  
### 🎬 Cinematic UI/UX  
- Smooth animations powered by Framer Motion  
- Modern bento-grid layout  
- Soft shadows, rounded cards, and clean spacing  
### 📅 Smart Daily Management  
- Track tasks with an intuitive interface  
- Organized daily workflow structure  
- Focus on clarity and minimal cognitive load  
### ✅ Task Management System  
- Full CRUD operations (Create, Read, Update, Delete)  
- Persistent storage with MongoDB  
- Clean and responsive task handling  
### 📝 Notes / Clipboard  
- Capture quick thoughts and ideas  
- Glass-morphism inspired design  
- High readability and distraction-free writing  
### 🔐 Secure Authentication  
- JWT-based login & signup system  
- Password encryption using Bcrypt  
- Protected routes and user-specific data  
---
## 🎨 Design Language  
DearDays follows a calm and aesthetic design philosophy to enhance focus:
- **Primary Color:** Deep Blue – Structure & clarity  
- **Accent Color:** Warm Orange – Actions & highlights  
- **Secondary Tone:** Soft Blue – Interaction states  
- **Background:** Light Cream – Reduced eye strain  
---
## 🛠️ Tech Stack  
### Frontend  
- React.js  
- Tailwind CSS  
- Framer Motion  
- Axios  
### Backend  
- Node.js  
- Express.js  
- MongoDB  
- JWT Authentication  
- Bcrypt

## 🚀 Getting Started  
### 1. Clone the Repository  
```bash
git clone <your-repo-link>
cd DearDays

2. Install Dependencies

Backend

cd backend
npm install
npm run dev

Frontend

cd frontend
npm install
npm start

3. Environment Variables

Create a .env file inside backend:

PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
